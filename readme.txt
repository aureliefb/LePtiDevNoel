Site développé en déc 2022 en réponse à un concours lancé par un groupe de développeurs.
Thème et consignes : créer, sans template, un site web sur le thème de Noël, tous langages et technos autorisés.
Date limite : 31 décembre 2022.
Prix : 1 an d'hébergement chez O2Switch

------------

Ce concours m'a permis de me challenger et de voir ce que j'étais capable de faire en quelques heures.
N'ayant pas d'hébergement, j'ai déposé les fichiers sur GitHub et publié le site sur GitHub Pages (qui ne prend pas en charge les fichiers .php, donc le site n'est pas fonctionnel).

4 participants, 3 gagnants dont je fais partie \o/
